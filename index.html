L² error vs N
L² error = √∫(u_N − u_ref)² dx over x∈[0,1] at fixed t. Measures average error across the whole domain. A steep drop means each new term contributes significantly. Flat section means those terms are negligible at this t — try a smaller t to see more variation.
L∞ error vs N (max pointwise)
L∞ error = max over x of |u_N(x) − u_ref(x)|. Worst-case error at any single point. Stricter than L² — catches localised spikes that L² averages away. If L² and L∞ drop at the same rate, error is evenly spread across x (smooth φ). If L∞ drops slower, there is a spike somewhere — look at the pointwise error plot.
absolute pointwise error |u_N(x) − u_ref(x)| over x
Each line = error at every x for one N, on a log scale. Light gray = N=1, black = N=max. Sharp dips to near-zero are where u_N and u_ref cross (difference = 0, log→−∞). Spikes at x=0 and x=1 collapsing to 10⁻²¹ = boundary condition satisfied exactly (sin(nπ·0)=sin(nπ·1)=0). Overall level of each line dropping as N increases confirms convergence across the whole domain.
error table
L² and L∞ errors as absolute values. Reduction = error(N=1) − error(N), same units as error. u_max and u_min show the range of the N-term solution — confirms whether the solution shape is physically reasonable.
φ(x) vs u(x, t) — initial condition vs evolved solution
Dotted = φ(x), the initial temperature at t=0. Solid = u(x,t) at chosen t using all N terms. Dashed = reference. The gap between φ(x) and u(x,t) shows how much heat has diffused by time t. At small t the lines are close; at large t u(x,t) flattens toward zero as heat dissipates.
Aₙ coefficient magnitude
|Aₙ| = |2∫φ(x)sin(nπx)dx| on a log scale. Weight of the n-th mode in the series. Fast decay (steep slope) → series converges in few terms. Slow decay (shallow slope) → many terms needed. Zero coefficients for even n means φ(x) is symmetric about x=0.5. The slope approximates the convergence rate: slope ≈ −p means |Aₙ| ~ 1/nᵖ.
